# Objektno usmerjeno programiranje & dekoratorji v Pythonu

Učni zvezek (Jupyter Notebook) za osnove objektno usmerjenega programiranja (OOP) in dekoratorjev v Pythonu.

Zvezek je nastal ob gledanju tečaja **[Bro Code](https://www.youtube.com/@BroCodez)** na YouTubu.

## Vsebina

- **Dekoratorji** — ovojnica, `*args`/`**kwargs`, `@` sintaksa
- **Konstruktor & metode** — `__init__`, atributi, metode
- **Razredne spremenljivke** — razlika med instančnimi in razrednimi spremenljivkami
- **Dedovanje** — enkratno, večkratno, večnivojsko
- **Abstraktni razredi** — `ABC`, `@abstractmethod`
- **`super()`** — razširjanje podedovanih metod
- **Polimorfizem** — isti vmesnik, različne implementacije
- **Duck typing** — polimorfizem brez dedovanja
- **Agregacija & kompozicija** — razmerja med objekti
- **Gnezdeni razredi** — grupiranje sorodnih razredov
- **Statične metode** — `@staticmethod`
- **Razredne metode** — `@classmethod`
- **Dunder metode** — `__str__`, `__eq__`, `__lt__`, `__add__`, `__contains__`, `__getitem__`
- **`@property`** — getter, setter, deleter
